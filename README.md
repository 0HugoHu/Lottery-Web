## Usage

Edit ```js/member.js``` file for your own database:
```js
var member = [
  {
    "genre": "Fried Chicken",
    "name": "Bojangles"
  },
  {
    "genre": "Causual Mexican",
    "name": "Monterrey Mexican"
  },
  {
    "genre": "Tex-Mex cuisine",
    "name": "Chili's Grill"
  },
  ...
```
